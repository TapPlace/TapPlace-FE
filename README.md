## 개발환경

- FrontEnd

  언어 : JavaSciprt(TypeScript)

  라이브러리 : Redux, ReduxToolkit, Axios, SCSS, FontAwesome ...

## 프로젝트 구조

```bash
├── config # eject
├── public # index.html, Image Files
├── script # script
└── src
   ├── components # Components View
   ├── hooks # Custom Hook
   ├── img # Image Files
   ├── pages # Page View
   ├── redux # Redux File
   └── style # CSS File
```
